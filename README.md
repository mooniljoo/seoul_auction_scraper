## How to build

```
$ yarn
$ yarn package
$ cd out
```

## 사용법 (Usage)

1. 압축해제 후 생성된 폴더의 내부의 Auction Website Scraper.exe 실행.
2. 입력되어 있는 url 주소에서 'Enter' key를 입력하거나 'SUBMIT'버튼 click 으로 작동.
3. 1-2 초 내외로 자동화된 브라우저 실행, 스크래핑 작동, 프로그램 내부에 테이블 생성.
4. 정상 스크래핑 종료시 엑셀파일 생성.
